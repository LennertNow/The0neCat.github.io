Competenties

Ik kan al heel mijn leven goed tegen stress en ik kan deze gezonde stress ook onder controle houden om zo beter te werken wanneer nodig. Door elke situatie blijf ik rustig en kalm, voor elk probleem is een oplossing

Communicatie

Door mijn luisterend oor in combinatie dat ik me goed in andermans schoenen kan zetten zorgt het ervoor dat ik goed kan communiceren met de klant.

Doorgroeien

In alles wat ik doe wil ik altijd doorgroeien. Zo ben ik binnen de studentenvereniging van de functie Facbar naar Penningmeester gegaan. Hierbij heb ik ook beter leren multitasken en doe ik dus ook meerdere dingen tegelijk op een dagelijkse basis.

Mijn motto

Mijn motto is altijd: Behandel mensen zoals jezelf behandelt wilt worden. Op die manier ben ik dus altijd respectvol en lief tegen iedereen. Veel geduld heb ik ook wat wel vaak handig is.
